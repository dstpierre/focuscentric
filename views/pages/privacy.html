TL:DR : S.V.P., aidez-nous à rester en affaire en ne partageant pas nos productions payantes. Également, nous ne ferons rien de mal avec vos informations et certainement les distribuer a qui que ce soit.

Respecter nos efforts.

Nous travaillons fort pour créer des formations et nous pensons que le prix est justifié et abordable. Si vous n'êtes pas d'accord, veuillez nous le spécifier. Vous verrez que nous répondons très rapidement et dans certains cas nous pourrions revoir à la baisse le prix de certaines productions.

La licence de téléchargement

Lors de l'achat d'une formation, vous avez un accès à télécharger les vidéos pour une utilisation personnelle. Nous pouvons faire des prix de groupe également. Si vous êtes un groupe, veuillez nous contacter.

Vous pouvez diffuser une vidéo pour un « User Group » ou un « Dev Meeting », mais nous aimerions être avisé et si possible de faire passer le mot à vos membres.

Ce sont les ventes qui nous permettent de continuer à créer des formations. Nous vous demandons de ne pas partager ce que vous téléchargez. Merci.

Les retours

Nous sommes persuadés que vous allez apprécier et allez apprendre de nos formations. Si tel n'est pas le cas, nous vous rembourserons dans les 15 jours suivants votre achat.

Information privée

Vos informations sont importantes pour nous. Chez Focus Centric nous adhérons à ces principes fondamentaux :

Nous ne vous demanderons jamais d'information privée sans que nous en aillions vraiment de besoin.

Nous ne conservons pas vos informations de crédit sur nos serveurs. Seules les informations relatives aux opérations et à la navigation sont conservées.

Voici notre politique relative aux informations privées.

Focus Centric inc. qui opère le site web focuscentric.com. Focus Centric s'engage à protéger les informations que vous nous fournissez pour les opérations du site web.

Visiteur du site web

Comme la plupart des sites web Focus Centric recueil des informations non personnelles tel que la version et nom de votre navigateur, votre langue par défaut, le site web ayant référé votre visite, etc. Le but étant d'analyser comment les visiteurs du site web utilisent les pages pour mieux optimiser la navigation dans le future.

Cueillette d'information personnelle

Certains visiteurs du site web interagissent de sorte que nous devons récupérer des informations personnelles. Ces informations dépendent du type d'interaction désiré. Les informations financières lors d'une transaction ne sont pas conservées sur les serveurs de Focus Centric. La communication avec le service de prise de paiement se fait sous un lien sécurisé (SSL).

Vous avez accès en tout temps à modifier ou supprimer les informations liées à votre compte utilisateur Focus Centric.
